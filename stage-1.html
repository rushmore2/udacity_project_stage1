The Basics of the Web and HTML

What is A Web Page?

A webpage is a text document written in a language called HTML. HTML stands for hyper text mark up language which is the main document type of the web.

How the Web Works?

The web has four components - the Browser, Internet, HTTP (which is the main protocol of the web), and Servers. When a user open a web browser and connects to www.udacity.com by sending an HTTP request via internet, the server for www.udacity.com will then retrieves the document for the website and sends it back to the user's browser. HTML Vocabulary

The Basic HTML vocabulary consists of Tags, Elements and Attributes. Most Tags have an opening and a closing tag. Some tags (called "void" tags) do not require a closing tag. An HTML element refers to everything within a set of opening and closing tags. The Attribute is a property of an HTML element. This video will show you the structure of the HTML tags and elements as well as to make links using attributes.

Inline vs. Block Elements

HTML elements are either inline or block. Block elements create an "invisible box" around the content inside of them. a, span, br, tag and strong are all inline elements while div, p and form are block elements.

Creating a Structured Document with HTML

HTMLs "Tree-Like" Structure

A "tree-like" structure or tree diagram is a way of representing the hierarchical nature of a structure in a graphical form. It is named a "tree structure" because the classic representation resembles a tree, even though the chart is generally upside down compared to an actual tree, with the "root" at the top and the "leaves" at the bottom.

HTMLs vs. CSS

HTML and CSS are both "languages". HTML controls the "structure" of a web page (with syntax and rules) while CSS controls the "style" of a page (or how it looks).

Boxes Everywhere

Everything on the web are boxes. A programmer must first start out with understanding and cutting up the boxes - from the big box and split it up piece by piece and work from big to small.

Text Editors

A text editor is a computer program that allows you to create and edit files that contain plain text. This is very different than a robust word processing program like Microsoft Word or Google Docs, because they add special formatting that's hidden to the user.

Text editors are commonly used by web designers and developers because it provides maximum flexibility to create just about anything. This is where professionals "hand code" HTML, CSS, JavaScript, and many other languages. Text editors like Scratchpad.io and Codepen.io are great tools and best for small projects. But as your web page gets bigger, a program that you can keep on your computer like Sublime Text is recommended.

Adding CSS Style to HTML Structure

What is Cascading?

Cascading Style Sheets (CSS) is a style sheet language used for describing the look and formatting of a document written in a markup language.

Cascading means that rules are applied not only to the elements they directly match, but also to all of those elements' child elements. However, if a child element has multiple, overlapping rules defined for it, the more specific rule takes effect.

CSS and its Key Feature

Inheritance is a key feature in CSS, it relies on the ancestor-descendant relationship to operate. In general, descendant elements inherit text-related properties, but box-related properties are not inherited.

Avoiding Repetition

Inheritance prevents certain properties from being declared over and over again in a style sheet, allowing the software developers to write less CSS. It enhances faster-loading of web pages by users and enables the clients to save money on bandwidth and development costs.

One way to avoid repetition is by usng the HTML class attribute in div tag. The class attribute provides a way of classifying similar elements then style them together in CSS.
